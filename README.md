# EF Model First

Подход Model First в Entity Framework 6.

## Описание
Проект демонстрирует подход Model First:
- Визуальное проектирование модели
- Создание базы данных из модели
- Генерация кода из визуальной модели

## Технологии
- .NET Framework 4.7.2
- Entity Framework 6
- SQL Server
- ADO.NET Entity Data Model

## Особенности
- Модель создана визуально через EF Designer
- База данных сгенерирована из модели
- Автоматическая генерация классов и контекста
- Использование T4 templates для генерации кода

## Подход Model First
1. Создание визуальной модели
2. Генерация базы данных из модели
3. Генерация классов C# из модели
